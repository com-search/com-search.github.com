Computer Information Search Website

该界面为电脑信息搜索界面，在上面两个文本框中输入电脑品牌和型号，点击搜索按钮。

在位于下方的文本框中将显示该产品的价格，便于客户对产品的了解和购买。

通过界面的Login选项可以跳转到登录界面，以进入后台界面,对数据库内容进行修改。通过Homepage选项，可以进入项目介绍页面。


Computer Information Input Website

该界面为电脑信息存储界面，在上面两个文本框中输入电脑品牌和型号，在下面的文本框中输入该产品的价格，点击存储按键。

该品牌的该电脑型号及其价格将存储到数据库中，可在前端界面查找到。

通过界面的Goback选项课跳转回搜索界面，检查输入结果。通过Homepage选项，可以进入项目介绍页面。
